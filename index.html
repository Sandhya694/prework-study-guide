HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is short for "HyperText Markup Language"
  • HTML tags are the hidden keywords within a web page that define how your web browser must format and display the content.
  • The html tags defines the document as web page. It also identifies the beginning and end of the HTML document. All other tags must fall between the html tags.
  • The body tags contain all the information and other visible content on the page. All your images, links and plain text must go between the body tags.
  • The body element is an example of a semantic HTML element
  • Semantic HTML also plays an important role in making sure your webpage is accessible to the widest variety of users, including those who use assistive technologies such as screen readers
  • An element refers to the entire element from opening tag to closing tag, whereas a tag refers only to what appears inside the angle brackets.
  • The h1 through h6 elements represent the level of heading a given text block represents. Headings are exactly what they sound like: larger or more prominent titles and subtitles to organize a page
  • The biggest heading h1 should only appear once on each page.
  • The p element represents a paragraph or block of text
  • The ul, ol, and li elements represent unordered lists, ordered lists, and list items. In essence, these HTML elements represent bulleted or numbered lists
  • The img element contains information about images that are displayed on the webpage. It can contain various attributes; the src attribute is the most important because it defines the location of the image.
  • The br element creates a line of empty space, or a line break, between two blocks of content.
  • The hyperlink, the "a" element (which stands for anchor) creates links to the same webpage or other webpages. The href attribute points to the URL for the link
  • section elements are semantic elements that define a section in a document

CSS

  • CSS-Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language such as HTML
  • Three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
    • To style an element directly in the HTML file, they use inline CSS. Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only
    • Internal style sheet contains CSS rules for the webpage in the head section of the HTML file. The rules only apply to that page, and you can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in the page code.
    • The most common way to style a webpage is by using an external CSS style sheet. This is preferred in most cases because it allows developers to keep all their CSS rules in a separate file, which makes design changes easier.When using an external style sheet, we link the file using a link element placed in the HTML's head
  • CSS rules
    • selector defines the element or attributes to which the rules, or declarations, will apply.
    • Declarations contain two important components: the CSS property we want to apply and the value of the property. For example, the declaration color: blue; includes the CSS property color and the value for that property, blue.
    • A selector can have multiple declarations thenit is called as declaration block
  • CSS BOX MODEL
    • The content box-which is the area that the content lives in
    • The padding box-surrounds the content box and is the space created by the padding property
    • The border box-surrounds the padding box and its space is occupied by the border value
    • The final area, the margin box, is the space around your box, defined by the margin rule on your box
  • * is called universal selector any declaration included here will be applied to the whole page
  • We can combine two selectors into a single CSS rule with a comma, allowing the declarations to affect both selectors
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector
  • The period preceding class attribute name designates this as a class selector. Any HTML element assigned to the class, will gain these property assignments from the CSS

JAVASCRIPT

  • JavaScript is the programming language of the Web
  • JavaScript code is inserted between script tags. It can be placed in the body, or in the head section of an HTML page
  • javaScript files have the file extension .js.To use an external script, put the name of the script file in the src (source) attribute of a script tag
  • Multi-line comments start with /* and end with */.Any text between /* and */ will be ignored by JavaScript.
  • The cSemicolons separate JavaScript statements. Add a semicolon at the end of each executable statementonsole. log() method outputs a message to the web console
  • Variables are containers for storing data (storing data values). Variables, declared with the var,let or const keyword
  • Control flow is the order in which a computer executes code in a script,code runs from the first statement at the top of the page, down to the last statement at the bottom of the page
  • A JavaScript function is a block of code designed to perform a particular task.A JavaScript function is executed when "something" invokes it (calls it)
  • A JavaScript function is defined with the function keyword, followed by a name, followed by parentheses ().
  • Function names can contain letters, digits, underscores, and dollar signs (same rules as variables).
  • The parentheses may include parameter names separated by commas:(parameter1, parameter2, ...)The code to be executed, by the function, is placed inside curly brackets: {}
  • else-to specify a block of code to be executed, if the same condition is false
  • if - to specify a block of code to be executed, if a specified condition is true
  • else if - to specify a new condition to test, if the first condition is false
  • An array is a special variable, which can hold more than one value
  • for-Marks a block of statements to be executed in a loop
  • for (expression 1; expression 2; expression 3) { // code block to be executed }
  • Expression 1 is executed (one time) before the execution of the code block.Expression 2 defines the condition for executing the code block.Expression 3 is executed (every time) after the code block has been executed.
